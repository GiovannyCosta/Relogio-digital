# 🕒 Relógio Digital Dinâmico

Um relógio digital elegante que ajusta automaticamente a saudação (Bom dia, Boa tarde, Boa noite) e os ícones conforme o horário do sistema. Suporta múltiplos idiomas (PT-BR / EN-US).

## ✨ Funcionalidades
- **Hora em tempo real**: Atualização segundo a segundo.
- **Saudação Dinâmica**: Muda conforme o período do dia.
- **Internacionalização**: Fácil troca entre Português e Inglês.
- **Design Responsivo**: Interface limpa que se adapta a diferentes telas.

## 🚀 Tecnologias
- HTML5
- CSS3 (Variáveis e Flexbox)
- JavaScript Vanilla

## 🛠️ Como usar
Basta abrir o arquivo `index.html` em qualquer navegador.
